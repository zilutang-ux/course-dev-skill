# 课程研发设计 Skill（ADDIE × 大片课融合版）

> **课程的本质是「换脑」，唯有大片式课程才能实现深度认知改变**

## 方法论融合

| 方法论 | 角色 | 核心价值 |
|--------|------|---------|
| **ADDIE** | 流程骨架 | 分析→设计→开发→实施→评估（系统性+可追溯） |
| **大片课** | 内容质量 | 天龙八部（冲击力+换脑效果） |

## 课程四部曲（好莱坞映射）

| 好莱坞 | 课程 | 使命 |
|--------|------|------|
| 策划 | 课程准备 | 说清（定位+框架） |
| 编剧 | 课程内容 | 说服（开场+中场+收场） |
| 导演 | 课程演绎 | 说动（共情+悬疑） |
| 发行 | 课程销售 | 说销（钉子营销） |

顺序不可逆：说不清→无法说服→无法说动→无法说销

## 五阶段全景

```
阶段一：分析 × 定位 ──→ 阶段二：设计 × 框架 ──→ 阶段三：开发 × 内容
  (找钉子)                (传信任)               (写剧本)
                                                      │
                                                      ▼
阶段五：评估 × 销售 ←── 阶段四：实施 × 演绎 ←────────┘
 (闭环迭代)              (说动学员)
```

## 阶段一：分析 × 定位（找钉子）

**核心问题：** 给谁解决什么问题？
**使命：** 三天课程浓缩成一句话，像钉子钉入学员脑海

### 执行步骤
1. **学员调研**（≥10人访谈/问卷）→ 找痛点、找差距、找目标
2. **找钉子**（三选一路径）：
   - A. 从大痛点出发 → 定领域→定画像→定挑战→定观点
   - B. 从大师论出发 → 定领域→定画像→定观点→定挑战
   - C. 从大竞争出发 → 最佳实践独特感悟→与众不同
3. **输出钉子三要素：**

| 要素 | 公式 | 输出 |
|------|------|------|
| 一痛 | "你有病" | 一句话描述核心痛点 |
| 一招 | "我有药" | 一句话描述核心方法 |
| 一不 | "就我有" | 一句话描述差异化 |

### "一"的哲学
- **取一舍九：** 三天内容浓缩成一个点
- **一箭穿心：** 直触学员心灵，尖锐坚硬无坚不摧
- **一直敲：** 不断重复强调，至少1000次

### 定位 vs 主题（常见误区）

| 课程主题（❌不是定位） | 课程定位（✅钉子） |
|----------------------|------------------|
| 通用，谁都能用 | 独特，只有你 |
| 品牌战略、绩效管理 | 大片换脑、顺势借力 |

### 质量标准
- [ ] 钉子一句话说清，3秒内理解
- [ ] 同时满足：大师论 + 大挑战 + 大竞争
- [ ] 讲师有底气反复强调

> ⚠️ **没有钉子绝不进入下一阶段！**

### 讲师常见卡点（提前预判）
- 说不清 → 内容庞杂，未做减法
- 找借口 → "培训行业不需要定位"
- 没耐心 → 遇挫折就放弃
- 没决心 → 浅尝辄止，频繁换赛道

### 本阶段输出
📄 **课程定位文档**（含需求分析 + 钉子定义）

## 阶段二：设计 × 框架（传信任）

**核心问题：** 框架能否传递信任？
**底层逻辑：** 一个好框架 = 100个销售员

### 框架三重身份
- **专业标志：** 框架杂乱 = 虚有其表
- **治学态度：** 框架体现严谨程度
- **感召武器：** 先感动自己→感动伙伴→感动学员

### 执行步骤
1. **课程架构设计** — 拒绝纪录片式，采用大片故事线串联
2. **三把利剑检验：**
   - 一脉相承：每个模块与钉子关系清楚
   - 一剑封喉：去掉任一模块课程不完整
   - 一首诗词：框架朗朗上口
   - 进阶：环环相扣（模块之间逻辑关系+先后顺序）、严丝合缝（各模块关键动作高度一致）
3. **1/3/5分钟检验：**
   - 1分钟入眼（看框架图被吸引）
   - 3分钟入耳（听讲解被打动）
   - 5分钟入心（能复述并转介绍）
4. **评估方案设计** — 前测/形成性/总结性/跟踪评估

### 本阶段输出
📄 **课程设计文档**（含框架图 + 模块大纲 + 评估方案）

## 阶段三：开发 × 内容（写剧本）

**核心问题：** 能否让学员"决心改命→挑战现状→促使行动"？

### 90分钟课堂节奏

```
开场 15min（决心改命）→ 中场 50min（挑战现状）→ 收场 25min（促使行动）
     全程贯穿：共情 + 悬疑
```

### 开场三步曲（定义使命→愿景→标准）
- 使命：学员要实现的最高目标
- 愿景：实现后的美好画面
- 标准：重新定义行业高标准
- 开场即第一印象，决定学员全场认知

### 中场三步曲（揭示→挑战→对比现状）
- 揭示现状：具体描述旧认知/旧做法
- 挑战现状：分析旧做法为何失败（追问5层为什么）
- 标杆对比：最佳实践与现状的差距

### 收场三步曲（一招制胜→做标准→给信心）
- 一招制胜：狠练一招 + 抓牛鼻子
- 做标准：可操作动作+责任人+时间
- 给信心：亲身经历/学员成功案例
- **实效教学法四步：** 我做你看→我说你听→你做我看→你说我听
- 收场=兑现承诺，不促使行动=背信弃义

### 全场贯穿设计
- **共情四标准：** 真（讲我所做做我所讲）+ 爱（敬畏学员）+ 趣（讲故事高手）+ 梦（有追求有梦想）
- **悬疑三价值：** 制造注意力 + 制造思考力 + 制造说服力
- **悬疑两大标准：** 从头问到尾 + 从头应到尾

### 本阶段输出
📄 **完整课程产品**（课件 + 脚本 + 素材 + 评估工具 + 讲师手册）

## 阶段四：实施 × 演绎（说动学员）

**核心问题：** 讲师能否说动学员去行动？

### 执行步骤
1. **讲师磨课** — 重点训练共情和悬疑的演绎能力
2. **环境部署** — 技术/场地/设备就绪
3. **试点运行** — 小规模测试，收集反馈
4. **正式发布** — 全面铺开

### 每节课自检五问
- [ ] 开场让学员下定决心了吗？
- [ ] 中场成功挑战学员现状了吗？
- [ ] 收场学员有可执行的行动方案和信心吗？
- [ ] 学员感受到"你懂我"了吗？
- [ ] 学员全程保持投入和期待吗？

### 本阶段输出
📄 **讲师手册 + 学员指南 + 演绎自检表**

---

## 阶段五：评估 × 销售（闭环迭代）

**核心问题：** 有效果吗？学员愿意推荐吗？

### 双重评估体系

**ADDIE柯氏四级评估：**
| 层级 | 内容 | 时机 |
|------|------|------|
| L1 反应 | 满意度、参与度 | 课程结束 |
| L2 学习 | 知识/技能掌握 | 课程中/结束 |
| L3 行为 | 行为改变应用 | 课后1-3月 |
| L4 结果 | 业务绩效影响 | 课后3-6月 |

**大片课换脑八问：**
1. 定位清楚？ 2. 框架可信？ 3. 开场改命？ 4. 中场挑战？
5. 收场行动？ 6. 共情懂你？ 7. 全程投入？ 8. 愿意推荐？

### 数据驱动迭代
```
评估数据 → 问题诊断 → 回溯对应阶段 → 修改 → 再评估

L1低 → 回到Design | L2低 → 回到Development
L3低 → 回到Implementation | L4低 → 回到Analysis
```

### 钉子营销（概念框架）
定义价值 → 承诺价值（用成果兑现）→ 传播价值（学员自发转介绍）
> 注：完整方法论尚未公开发布，目前只有概念框架。不疯狂推销，不损害讲师威信。

### 本阶段输出
📄 **评估报告 + 迭代方案 + 下版本计划**

## 使用方法

### 开发新课程
```
用户：我要开发一门 [主题] 课程，目标学员是 [画像]，想解决 [问题]
```
→ 复述理解 → 进入阶段一找钉子 → 引导完成钉子三要素 → 进入阶段二

### 打磨现有课程
```
用户：我有一门 [课程名]，帮我打磨升级
```
→ 收集现有材料 → 用换脑八问诊断 → 输出问题清单 → 协助修改

### 课程评审
```
用户：帮我评审这门课的质量
```
→ 收集课程材料 → 按换脑八问打分 → 按柯氏四级评估 → 输出诊断报告
