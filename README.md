> 一套完整的课程研发设计流程，融合ADDIE体系化框架与大片课实战方法论，让每一门课都成为「大片级」学习体验。

## ✨ 核心理念

> **课程的本质是「换脑」，唯有大片式课程才能实现深度认知改变。**

本技能融合两大方法论：

| 方法论 | 角色 | 核心价值 |
|--------|------|---------|
| **ADDIE** | 流程骨架 | 分析→设计→开发→实施→评估 |
| **大片课** | 内容质量 | 天龙八部：定位→框架→开场→中场→收场→共情→悬疑→销售 |

## 🎯 适用场景

- 💼 商业课程打磨升级
- 🏢 企业内训课程开发
- 🎬 AI漫剧培训设计
- 🌟 知识IP打造

## 📋 五阶段全景

```
阶段一：分析 × 定位 ──→ 阶段二：设计 × 框架 ──→ 阶段三：开发 × 内容
  (找钉子)                (传信任)               (写剧本)
                                                      │
                                                      ▼
阶段五：评估 × 销售 ←── 阶段四：实施 × 演绎 ←────────┘
 (闭环迭代)              (说动学员)
```

### 阶段一：分析 × 定位
- 学员调研（≥10人深度访谈）
- 找钉子：「你有病，我有药，就我有」
- 钉子三要素：一痛、一招、一不

### 阶段二：设计 × 框架
- 拒绝纪录片式，采用大片式框架
- 三把利剑检验：一脉相承、一剑封喉、一首诗词
- 1/3/5分钟检验：入眼→入耳→入心

### 阶段三：开发 × 内容
- 大片开场：定义使命→愿景→标准（决心改命）
- 大片中场：揭示现状→挑战现状→标杆对比
- 大片收场：抓牛鼻子→做标准→给信心
- 全场共情 + 悬疑设计

### 阶段四：实施 × 演绎
- 讲师磨课SOP
- 每节课演绎自检

### 阶段五：评估 × 销售
- 柯氏四级评估
- 换脑八问评估
- 数据驱动迭代

## 📖 关键概念

### 钉子公式
> **「你有病，我有药，就我有」**
> - 一痛：学员最大的痛点
> - 一招：你的核心解决方案
> - 一不：与竞争对手的最大不同

### 换脑八问
1. 学员能一句话说清课程内容吗？（定位）
2. 学员能复述课程框架吗？（框架）
3. 学员听完开场有改变决心吗？（开场）
4. 学员承认旧认知被挑战了吗？（中场）
5. 学员有具体的行动方案吗？（收场）
6. 学员觉得"讲师懂我"吗？（共情）
7. 学员全程投入吗？（悬疑）
8. 学员愿意推荐/复购吗？（销售）

### 90分钟课堂节奏
```
开场 15min → 中场 50min → 收场 25min
（定义使命）  （挑战现状）   （促使行动）
     全程贯穿：共情 + 悬疑
```
