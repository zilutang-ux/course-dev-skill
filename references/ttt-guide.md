# TTT讲师培训指南

> Training the Trainer — 讲师能力培养与课程打磨方法

## 讲师能力三层模型

```
第一层：专业功底（你讲什么）
  → 你在这个领域的深度洞察和实践经验
  → 没有专业功底，技术再好也是空壳

第二层：授课技术（你怎么讲）
  → 大片天龙八部 = 八道工序
  → 共情 + 悬疑的演绎能力

第三层：个人魅力（你是谁）
  → 巅峰状态、信念感、感染力
  → 学员不仅学你的知识，更要成为你
```

## 讲师常见问题与对策

| 问题 | 根因 | 对策 |
|------|------|------|
| 说不清定位 | 没有钉子/钉子太多 | 找钉子，取一舍九 |
| 讲不深刻 | 没有追问到根本原因 | 追问5层为什么 |
| 讲太多招 | 不敢做减法 | 抓主要矛盾，只讲一招 |
| 没有共情 | 只关注内容不关注人 | 用学员的语言讲学员的故事 |
| 没有悬疑 | 平铺直叙 | 每个模块抛问题→给挑战→揭答案 |
| 不敢卖课 | 怕损害专家形象 | 钉子营销，不销而销 |

## 磨课SOP（10步诊断法）

```
Step 1: 听完整课程（全程录音/录像）
Step 2: 用换脑八问逐项评估
Step 3: 定位诊断 — 钉子清楚吗？
Step 4: 框架诊断 — 三把利剑通过吗？
Step 5: 开场诊断 — 能让人决心改命吗？
Step 6: 中场诊断 — 敢挑战学员现状吗？
Step 7: 收场诊断 — 有行动方案和信心吗？
Step 8: 共情诊断 — 学员觉得懂他吗？
Step 9: 悬疑诊断 — 学员全程投入吗？
Step 10: 输出修改建议 → 讲师修改 → 再磨
```

## 每节课演绎自检

- [ ] 开场让学员下定决心了吗？
- [ ] 中场成功挑战学员现状了吗？
- [ ] 收场学员有可执行的行动方案吗？
- [ ] 学员感受到"你懂我"了吗？
- [ ] 学员全程保持投入和期待吗？

## 实施准备清单

- [ ] 讲师磨课完成（重点训练共情和悬疑）
- [ ] 技术环境/场地/设备就绪
- [ ] 学员通知和学习指南发出
- [ ] 试点运行完成并调整
