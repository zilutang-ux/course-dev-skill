# 课程开发模板集

> 可直接使用的模板和检查清单

## 开场模板（决心改命）

```
你面对的最大使命是 [使命]。

如果做到了，你将 [愿景]。

但现实的标准是 [高标准]，你现在离这个标准还有 [差距]。

今天这堂课，我们一起解决这个问题。
```

### 开场使用要点
- 使命：学员要实现的最高目标
- 愿景：实现后的美好画面（可用故事/案例/数据）
- 标准：重新定义行业高标准
- 差距：让学员意识到现状与标准的距离

---

## 中场模板（挑战现状）

```
大多数人的做法是 [现状]。

但这带来了一个问题： [具体问题]。

为什么会这样？因为 [根本原因]。

而最佳实践告诉我们 [标杆对比]。
```

### 中场使用要点
- 现状：具体描述旧认知/旧做法，不能空泛
- 问题：旧做法带来的具体后果
- 根本原因：追问到最底层（追问5层为什么）
- 标杆对比：用自身最佳实践案例

---

## 收场模板（促使行动）

```
解决方案就是 [绝招]。

具体怎么做？
第一步 [动作1]
第二步 [动作2]
第三步 [动作3]

我自己 [亲身经历/案例]，你们一定也可以。

现在就去做。
```

### 收场使用要点
- 绝招：聚焦1个核心方法，不要讲太多
- 动作：可操作、可检查、有责任人、有时间
- 经历：用自己的成功故事给予信心
- 结尾：直接号召行动

## 钉子定位模板

```
课程名称：_______________
目标学员：_______________

一痛（你有病）：_______________
一招（我有药）：_______________
一不（就我有）：_______________

钉子一句话：_______________
```

### 钉子检验标准
- [ ] 一句话说清，任何人3秒内能理解
- [ ] 满足大师论（讲师独到见解）
- [ ] 满足大挑战（学员普遍痛点）
- [ ] 满足大竞争（与主流不同）
- [ ] 讲师自己有底气反复强调

---

## 课程框架模板

```
主线：[一句话钉子]

模块1：[决心改命]
  → 子主题1.1
  → 子主题1.2

模块2：[挑战现状]
  → 子主题2.1
  → 子主题2.2

模块3：[促使行动]
  → 子主题3.1
  → 子主题3.2
```

### 框架检验标准（三把利剑）
- [ ] 一脉相承：每个模块与钉子的关系清楚
- [ ] 一剑封喉：去掉任一模块课程不完整
- [ ] 一首诗词：框架朗朗上口

### 框架检验标准（时间检验）
- [ ] 1分钟入眼：给人看框架图，一看就被吸引
- [ ] 3分钟入耳：口头讲解框架，一听就被打动
- [ ] 5分钟入心：对方能自己复述并转介绍
